# 前缀和优化

这个应该是最简单的一个一种 DP\text{DP}DP 优化吧。

当 DP\text{DP}DP 过程中需要反复从一个求和式转移的话，可以先把它预处理一下，运算一般满足可减性。

先搞个简单的练练手，熟悉一下吧。

逆序对数列

我们设 f[i][j]f[i][j]f[i][j] 表示 iii 的全排列中，逆序对数量为 jjj 的个数。

那么我们只需要考虑在 i−1i-1i−1 的排列中插入 iii ，会增加多少逆序对数，那再设 kkk 为增加的逆序对数 (这里的 kkk 与题目中的 kkk 不一样), 就可以得到一下转移：

f[i][j]=∑k=0min(i−1,j)f[i−1][j−k]f[i][j]=\sum^{min(i-1,j)}_{k=0} f[i-1][j-k]f[i][j]=k=0∑min(i−1,j)​f[i−1][j−k]

或者可以写成这样

f[i][j]=∑k=max(j−i−1,0)jf[i−1][k]f[i][j]=\sum^{j}_{k=max(j-i-1,0)}f[i-1][k]f[i][j]=k=max(j−i−1,0)∑j​f[i−1][k]
