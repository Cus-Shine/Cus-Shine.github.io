# 前缀和优化

这个应该是最简单的一个一种 DP\text{DP}DP 优化吧。

当 DP\text{DP}DP 过程中需要反复从一个求和式转移的话，可以先把它预处理一下，运算一般满足可减性。

先搞个简单的练练手，熟悉一下吧。

# 逆序对数列

我们设 f[i][j]f[i][j]f[i][j] 表示 iii 的全排列中，逆序对数量为 jjj 的个数。

那么我们只需要考虑在 i−1i-1i−1 的排列中插入 iii ，会增加多少逆序对数，那再设 kkk 为增加的逆序对数 (这里的 kkk 与题目中的 kkk 不一样), 就可以得到一下转移：

f[i][j]=∑k=0min(i−1,j)f[i−1][j−k]f[i][j]=\sum^{min(i-1,j)}_{k=0} f[i-1][j-k]f[i][j]=k=0∑min(i−1,j)​f[i−1][j−k]

或者可以写成这样

f[i][j]=∑k=max(j−i−1,0)jf[i−1][k]f[i][j]=\sum^{j}_{k=max(j-i-1,0)}f[i-1][k]f[i][j]=k=max(j−i−1,0)∑j​f[i−1][k]

# 木棍分割

对于第一问还是很好做的，直接二分找出即可。对于第二问，我们可以想到设 f[i][j][k]f[i][j][k]f[i][j][k] 表示前 iii 根木棍 分成 jjj 组，且最长的一组长为 kkk。但会发现这 1≤k≤mid1 \leq k\leq mid1≤k≤mid，（这里的 midmidmid 就是第一问的答案）我们只需要在转移的时候让 kkk 合法即可。所以就可以列出以下转移:

f[i][j]=∑l=ki−1f[l][j−1]这里的k是满足summ[i]−summ[k]≤mid的最小的位置f[i][j]=\sum^{i-1}_{l=k}f[l][j-1] \\ \text{这里的 $k$ 是满足$summ[i]-summ[k]\leq mid$的最小的位置}f[i][j]=l=k∑i−1​f[l][j−1]这里的 k 是满足summ[i]−summ[k]≤mid的最小的位置

观察式子，发现每次都是加上一段连续的序列，那么我们可以用前缀和优化啦。

记 sum[i][j]=∑k=0if[k][j]sum[i][j]=\sum^{i}_{k=0}f[k][j]sum[i][j]=∑k=0i​f[k][j]

那么转移方程就又变成了这样: f[i][j]=sum[i][j−1]−sum[k−1][j−1]f[i][j]=sum[i][j-1]-sum[k-1][j-1]f[i][j]=sum[i][j−1]−sum[k−1][j−1]

发现 jjj 这一维是只与该层以及上一层有关，直接滚掉 jjj 这一维。

然后你根据以上思路交上代码会发现还有 TLE 的点，那是因为我们每次对与一个 iii 都在找满足 summ[i]−summ[k]≤midsumm[i]-summ[k]\leq midsumm[i]−summ[k]≤mid 的最小的位置，这样很容易超时，所以我们用 pre[i]pre[i]pre[i] 记一下对于位置 iii 所对应的 kkk 的位置。

那么这道题目就解决啦。
✔️success

# SAO

!! 这个题好 SAO！！

这种带有前置性的，并且根据数据范围能推出是一个树的，大概率是要用拓扑序了。

这个题其实就是让我们求解有多少种拓扑序而且题目中边方向的限制其实就是在限制拓扑序的前后，而且一般这种题目在设置 dpdpdp 状态的时都会考虑将拓扑序放到状态里，不然拓扑序就很难限制。

设 f[i][j]f[i][j]f[i][j] 表示以 iii 为根的字数， iii 在拓扑序中排第 jjj 位的方案书，那么转移显然是合并子树，即:

f[u][i+j]+=f[u][i]×f[v][k]×Cf[u][i+j]+=f[u][i]\times f[v][k]\times Cf[u][i+j]+=f[u][i]×f[v][k]×C

因为我们不在乎除了 uuu 和 vvv 之外的点是怎么排的，所以得乘个组合数。考虑当合并子树时，对于 uuu 的拓扑序，前面的 i−1i-1i−1 个肯定还是存在的，那么当 vvv 这个子树合并进来的时候，他的前面可能增加了 jjj 个，所以我们要从合并之后的 uuu 所在位置 i+ji+ji+j 的前面的 i+j−1i+j-1i+j−1 个位置中选出 i−1i-1i−1 个位置，那么他的后面也是同理，我们就可以得到 CCC 啦。

C=Ci+j−1i−1×Csiz[u]+siz[v]−i−jsiz[u]−iC=C_{i+j-1}^{i-1}\times C_{siz[u]+siz[v]-i-j}^{siz[u]-i}C=Ci+j−1i−1​×Csiz[u]+siz[v]−i−jsiz[u]−i​
